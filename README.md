# April Bouquet 💐

**April Bouquet** adalah katalog web premium berbasis Single Page Application (SPA) yang dirancang secara khusus untuk menampilkan koleksi bouquet secara mewah, elegan, dan profesional. Website ini dibangun tanpa framework (Pure HTML, CSS, JavaScript) dengan mengedepankan performa cepat dan desain antarmuka *Glassmorphism*.

## Fitur Utama ✨

- **Desain Ultra-Premium**: Menggunakan palet warna pastel eksklusif, efek kaca (*Glassmorphism*), dan animasi partikel dinamis.
- **Single Page Application (SPA)**: Transisi antar halaman (Katalog & Testimoni) menggunakan efek *slide* yang mulus tanpa memuat ulang browser.
- **Sistem Pencarian Pintar (Live Search)**: Mencari lebih dari 50 koleksi bouquet secara *real-time*.
- **Pemesanan WhatsApp Dinamis**: Dilengkapi dua opsi tombol:
  - 💬 **Tanya Detail**: Menyusun pesan WhatsApp khusus untuk bertanya kustomisasi produk.
  - 🛒 **Beli Sekarang**: Menyusun pesan WhatsApp khusus untuk langsung memesan produk.
- **Fitur Wishlist (Favorit Saya)**: Pengguna dapat menyimpan bouquet favorit mereka langsung di memori browser (Local Storage) melalui ikon hati (❤️).
- **Mode Gelap (Dark Mode) Otomatis**: Mendukung peralihan ke mode gelap elegan yang preferensinya akan tersimpan secara otomatis.
- **Toast Notifications**: Pemberitahuan interaktif bergaya aplikasi saat menyimpan produk.

## Teknologi yang Digunakan 💻
- **HTML5** (Semantik dan struktur halaman)
- **CSS3** (Variabel kustom, Flexbox/Grid, Animasi Keyframes, Media Queries)
- **Vanilla JavaScript** (Manipulasi DOM, Local Storage, Logika Filter/Pencarian)
